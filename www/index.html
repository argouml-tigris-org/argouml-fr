Bienvenue à ArgoUML

NOUVEAU! - UML 1.4, Diagrammes de Séquence, et support d'AndroMDA. Lire l'annonce de Fév. 2006.

ArgoUML est à la pointe des ateliers de modélisation UML open source et inclus le support de tous les diagrammes du standard UML 1.4. Il fonctionne sur toute plateforme Java et est disponible en sept langues. Voir la liste des fonctionnalités pour plus de détails.

ArgoUML a été téléchargé un demi-million de fois durant l'année 2005 et est utilisé dans le monde entier.

Lancer ArgoUML Maintenant (deux méthodes)

Lancer viaJava Web Start:
1. Cliquer pour lancer la dernière version stable.

Java Web Start est inclus dans les environnements d'exécution Java récents. Si le lien ci-dessus ne fonctionne pas, vous pouvez mettre à jour votre version de Java ici ou obtenir d'autre versions de Java Web Start et des informations sur la  page produit. Si vous avez des problèmes, merci de regarder la partie JWS de la FAQ.

Télécharger les fichiers et lancer:
1. Télécharger la dernière version stable: 0.20 zip (autres formats & modules additionnels)
2. Dézipper l'archive
3a. Sous Windows ou Mac: Double-cliquer sur argouml.jar
3b. Sous Unix: Taper la commande 'java -jar argouml.jar'

Capture d'écran (cliquer pour le tour)

L'interface utilisateur d'ArgoUML est divisée en 4 parties:
Supérieure gauche: une vue hiérarchique du fichier projet courant.
Supérieure droite: éditeur pour la partie séléctionnée du projet, dans ce cas un diagramme de classes.
Inférieure gauche: la liste des "penses-bêtes" du concepteur.
Inférieure droite: détails de l'objet sélectionné dans le diagramme ou du "pense-bête" sélectionné.

Diagrammes supportés:

  • Classes
  • Etats
  • Cas d'utilisation
  • Activité
  • Collaboration
  • Déploiement
  • Séquence

Pour les Développeurs d'ArgoUML

ArgoUML n'est pas seulement un outil de modélisation UML gratuit, c'est aussi un Développement Open Source auquel vous êtes invités à contribuer.

Allez dans la Developer Zone pour en lire plus sur le project et trouver de la documentation sur les entrailles d'ArgoUML.
